# 🎮 Своя Гра (Jeopardy) | Real-Time Web Engine

Повноцінний інтерактивний веб-рушій для проведення вікторин у форматі "Своя Гра" (Jeopardy). Проєкт розроблено на базі **Django** та **Django Channels** для забезпечення миттєвої синхронізації між дошкою питань, панеллю ведучого та смартфонами гравців (баззерами) через WebSockets.

## ✨ Головні фічі

*   **Real-Time Баззери (WebSockets):** Миттєва реакція на натискання кнопки гравцем. Глобальне блокування (хто перший натиснув — той і відповідає, кнопки інших блокуються).
*   **Синхронізація балів:** Нарахування або віднімання балів ведучим автоматично і миттєво відображається на екранах відповідних команд.
*   **Панель Ведучого (Admin Panel):** Зручний інтерфейс для контролю гри. Ведучий бачить, хто натиснув кнопку першим, може зараховувати бали та автоматично скидати раунд для наступного питання.
*   **Фінальний раунд:** При старті фіналу кнопки гравців автоматично перетворюються на поля для вводу тексту. Відповіді миттєво відправляються на екран ведучого.
*   **Захист від дублювання:** Команда може бути залогінена лише з одного пристрою одночасно.
*   **Mobile-First для гравців:** Інтерфейс баззера максимально оптимізований для швидкого натискання зі смартфонів.

## 🛠 Стек технологій

*   **Backend:** Python 3, Django, Django Channels (ASGI)
*   **Frontend:** HTML5, CSS3, Vanilla JS (WebSocket API)
*   **База даних:** SQLite (за замовчуванням)
*   **Дані гри:** Парсинг питань з локального `.json` файлу.

## 🚀 Встановлення та запуск (Локальна мережа)

Щоб грати з друзями в одній Wi-Fi мережі, виконайте наступні кроки:

```bash
git clone https://github.com/tiulenchik2/SvoyaGra-V2-Django.git
cd SvoyaGra-V2-Django

# Створення віртуального середовища
python -m venv venv
source venv/bin/activate  # Для Linux/macOS
# venv\Scripts\activate   # Для Windows

# Встановлення залежностей (переконайтеся, що є requirements.txt)
pip install -r requirements.txt
```
